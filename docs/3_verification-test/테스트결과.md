# 검증 테스트 결과

## 테스트 환경
- 테스트 파일: 1GB 바이너리 (`/dev/urandom` 생성)
- 실행: `docker compose up --build`

---

## OOM 검증

전송 중 `docker stats --no-stream` 측정 결과:

| 컨테이너 | MEM USAGE | MEM % | HOST 메모리 |
|---|---|---|---|
| grpc-file-streamer-server-1 | 53.23 MiB | 0.67% | 7.75 GiB |

**결론**: 1GB 파일 전송 중 서버 메모리가 53MB로 일정하게 유지됨. 청크 단위 스트리밍 I/O 정상 동작 확인.

---
